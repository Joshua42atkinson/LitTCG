## What Is LitTCG?

LitTCG is a game where kids spell real English words to summon collectible 3D creatures. Each creature's element, role, stats, and facial expression are derived from the word's etymology and psycholinguistic profile. Players use their word-pets in tactical battles against "Typos" and complete grammar-based quests for NPCs.

## The Hook

> *"Pokémon, but every creature is forged from a word you spell."*

## Key Features

- **9,582-word embedded curriculum** with real psycholinguistic data
- **38,400 unique emotive states** via the FACES protocol
- **Three-class RPS balance:** Slime (Logic), Golem (Grammar), Robot (Rhetoric)
- **Procedural 3D pet generation:** no hand-modeled assets required
- **Local-first, COPPA-safe:** no cloud accounts, no tracking
- **Cross-platform:** desktop today, web demo now, Android XR next

## Developer Bio

Joshua Atkinson is building LitTCG as a homeschool-friendly literacy tool that feels like a real game. The project combines classical Trivium concepts (Grammar, Logic, Rhetoric) with modern game design and psycholinguistics research.
